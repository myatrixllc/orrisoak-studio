Go to the API Keys section
In the left sidebar menu of the console, look for and click "API Keys". This is where you create and manage your keys.
5
Click "Create Key" and name it

Your full API key will appear on screen. It looks like sk-ant-api03-abc123xyz456...

Step 3 · Takes 3 Minutes · The Only "Technical" Step

Paste Your Key
Into the File

Open the file like a text document, find one specific spot, paste your key, and save. That's all.

🔧 Don't Be Scared — This Is Just Copy & Paste You are going to open the HTML file in a simple text editor (Notepad or TextEdit — programs already on your computer), find a placeholder text, replace it with your key, and save. It takes about 2 minutes. No coding involved.
If You're on Windows
1
Right-click the file and open with Notepad
Find orrisoak-marketing-agent.html in your folder. Right-click on it → move your mouse over "Open with" → click "Notepad". A window full of code will open. This is normal — don't change anything yet.
2
Open Find & Replace with Ctrl+H
Hold the Ctrl key and press H at the same time. A small "Find and Replace" window opens.
3
Type in the Find box
Click inside the "Find what:" box and type exactly this:
YOUR_ANTHROPIC_API_KEY_HERE
(all caps, with underscores, no spaces)
4
Paste your key in the Replace box
Click inside the "Replace with:" box and paste your API key (the sk-ant-... one you saved in Step 2). Use Ctrl+V to paste.
5
Click Replace All, then save
Click "Replace All". Notepad will say "1 replacement made." Then press Ctrl+S to save the file. Close Notepad. Done!
If You're on a Mac
1
Right-click and open with TextEdit
Find the file. Right-click → "Open With" → "TextEdit". If you see a warning about the file format, click "Edit Anyway." If the text looks strange, click the menu bar: Format → Make Plain Text, then try again.
2
Open Find & Replace
Press Command (⌘) + Option + F at the same time. A Find & Replace bar appears at the top or bottom of the screen.
3
Find the placeholder and replace with your key
In the Find field, type: YOUR_ANTHROPIC_API_KEY_HERE
In the Replace field, paste your API key with Command+V
Click Replace All.
4
Save the file
Press Command+S to save. Close TextEdit. Done!
What It Should Look Like Before & After
BEFORE — what's in the file originally
'x-api-key': 'YOUR_ANTHROPIC_API_KEY_HERE',
AFTER — once you paste your real key in
'x-api-key': 'sk-ant-api03-abc123XYZdef456GHIjkl789...',
⚠️ Keep Your Key Private Your API key is like a credit card number for the AI service. Don't share the HTML file publicly with your key inside it. If someone gets it, they can use your Anthropic credits. Only share the file with people you fully trust, like your own team.
✅ Done When: You open the file in Notepad and search for YOUR_ANTHROPIC_API_KEY_HERE — if you get "0 results found," it means your key was successfully replaced.

& Fixes

Problem: The AI doesn't respond at all when I type
Fix 1: Open the HTML file in Notepad and press Ctrl+F (or Cmd+F on Mac). Search for sk-ant-. If you don't find it, your key wasn't saved — go back to Step 3.
Fix 2: Make sure you clicked "Save" (Ctrl+S) after pasting the key in Notepad.
Fix 3: Log into console.anthropic.com/billing and make sure you have credit balance.
!
Problem: I see "Connection issue" or "error" message
Fix 1: Check you have internet — try loading Google.com in a new tab.
Fix 2: Your API key may have a typo. Go to console.anthropic.com, create a brand new key, and replace the old one in the file using Step 3 instructions again.
